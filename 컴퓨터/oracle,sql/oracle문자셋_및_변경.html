오라클 DB 문자셋과 언어셋 변경

Hare 2012-01-02 21:27:15 주소복사
조회 604  스크랩 1

1. 오라클 데이터베이스 문자셋과 언어셋 변경

 

■ 문자셋(CHARACTER SET) 변경
관리자로 접속하여 NLS_CHARACTERSET, NCHAR의 CHARACTERSET에 한국어를 지원하도록 파라미터의 속성값을 KO16MSWIN949로 변경.

 

 [문자셋 변경]
SQL>update sys.props$ set value$='KO16MSWIN949' 

        where name='NLS_CHARACTERSET';  

1 row updated.

SQL>update sys.props$ set value$='KO16MSWIN949' 

        where name='NLS_NCHAR_CHARACTERSET';

1 row updated. 


 

 

■ 언어셋(LANGUAGE SET) 변경
문자셋과 마찬가지로 오라클 데이터베이스 관리자로 접속하여 NLS_LANGUAGE 파라미터의 속성값을 AMERICAN_AMERICA.KO16MSWIN949로 변경.

 

[언어셋 변경]
SQL> update sys.props$ set value$='AMERICAN_AMERICA.KO16MSWIN949'

         where name='NLS_LANGUAGE';   
1 row updated.

 

[변경사항 저장 및 데이터베이스 재연동]

SQL> commit;
Commit complete.

 

SQL> shutdown
Database closed.
Database dismounted.
Oracle instance shut down.

 

SQL>startup
ORACLE instance started.

 

[변경사항 확인]

SQL> select * from v$nls_parameters;

 

2. 오라클 언어 환경변수 변경
오라클을 설치할 때 지정해 주었던 .bash_profile 파일에서 Oracle 언어 환경변수 설정

export NLS_LANG=AMERICAN_AMERICA.KO16MSWIN949

 

이렇게 해도 언어가 변경이 안되거나 물음표가 뜨는 경우는 dbca를 실행해서 DB를 밀고 다시 생성해서 언어셋을 설정하자.

또 혹시 이것도 안되면 오라클를 다시 설치해서 해야 할듯보인다.
